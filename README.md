# Interactive Cricket Game

This is a simple interactive cricket game built using HTML, CSS, and JavaScript. The user can simulate batting and bowling, and the game keeps track of runs, wickets, overs, and innings.

## Features

Two innings gameplay (Team A and Team B)
Randomized batting and bowling outcomes
Scoreboard updates in real-time
Commentary display
Dynamic target score for Team B
Match result handling
Option to restart the match

## How to Play

1. **Team A bats first.** Use the **Bat** button to score runs.
2. Each click simulates one ball.
3. When all overs or wickets are done, Team B begins batting.
4. Team B will chase the target score.
5. Game ends automatically when either team wins.
6. Click **Restart Match** to play again.
